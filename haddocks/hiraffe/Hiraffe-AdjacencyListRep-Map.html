Hiraffe.AdjacencyListRep.Map

Description

A Representation of graphs using adjacency lists. The adjacency lists are stored using a Map.

Synopsis

  • newtype GGraph f i v e = Graph (NEMap i (VertexData f i v e))

type Graph = GGraph Seq Source #

A graph in which the adjacency lists are stored using Data.Sequence

newtype GGraph f i v e Source #

A directed graph, whose vertices store data of type v, and are indexed by type i, and whose edges store data of type e. Edges are indexed by an ordered pair of vertex indices.

The ajdacency lsits are stored using the given f structure.
