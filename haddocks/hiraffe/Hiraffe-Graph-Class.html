class (HasVertices graph graph, HasEdges graph graph, BidirGraph_ graph) => Graph_ graph where Source #

A class representing non-empty undirected graphs. Note that every undirected graph is also a directed graph.

fromAdjacencyLists :: (Foldable1 f, Functor f, Foldable h, Functor h, vi ~ VertexIx graph, v ~ Vertex graph, e ~ Edge graph, GraphFromAdjListExtraConstraints graph h) => f (vi, v, h (vi, e)) -> graph Source #

Build a graph from its adjacency lists.

If the, in the list of neighbours of vertex u we see a vertex v that itself does not appear in the adjacencylist, we may drop it. In other words if u has a neighbour v, then v better have a specification of its neighbours somewhere.
