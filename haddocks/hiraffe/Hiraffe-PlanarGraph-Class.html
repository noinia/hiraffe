boundaryDartOf :: FaceIx planarGraph -> IndexedLens' (DartIx planarGraph) planarGraph (Dart planarGraph) Source #

Gets a dart bounding this face. I.e. a dart d such that the face lies to the left of the dart.

boundaryDartsFrom :: DartIx planarGraph -> IndexedFold1 (DartIx planarGraph) planarGraph (Dart planarGraph) Source #

Given some dart d, compute the darts along the face containing d (starting with d). I.e. we iterate nextDartOf until we are back at the starting dart.

Note that - if the dart is on the outerBoundary of a face, this reports the darts in CCW order - if the dart is on an an inner component, this means we report the darts in CW order.
